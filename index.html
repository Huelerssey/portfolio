Web Scraping imóveis

Este projeto é uma solução completa, que automatiza a extração de dados da web, realiza o tratamento e os insere em um banco de dados Cloud. Em seguida, é apresentado um modelo de previsão de preços utilizando técnicas de Machine Learning. Além disso, desenvolvi um Dashboard no Power BI para visualização de dados (DataViz) e por fim, uma aba de Storytelling para insigths e recomendações.

MANCHA CRIMINAL DE SÃO PAULO

Este projeto consiste em uma análise abrangente, que engloba desde a extração até a visualização dos dados. Utilizando técnicas de ETL e armazenando os dados em um banco cloud, desenvolvi uma Mancha Criminal a partir dos dados de geolocalização dos celulares, utilizando a biblioteca Geopandas em Python. Além disso, criei um dashboard no Power BI para proporcionar uma visualização clara e intuitiva dos dados analisados. Por fim, para compartilhar os resultados obtidos, desenvolvi um storytelling envolvente, apresentando de forma narrativa as descobertas e insights extraídos da análise.
